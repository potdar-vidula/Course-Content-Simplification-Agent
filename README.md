# 📚 Course Content Simplification Agent

> An AI-powered educational assistant developed using **IBM watsonx Orchestrate** and **IBM Granite Foundation Models** to simplify complex academic content into clear, student-friendly explanations.

---

## 📖 Project Overview

Understanding textbooks, research papers, and lecture notes can be challenging due to technical terminology and lengthy explanations. This project provides an AI-powered Course Content Simplification Agent that transforms complex educational content into simple, easy-to-understand language while preserving the original meaning.

---

## 🎯 Problem Statement

Many students, especially first-generation learners and those from non-English-medium backgrounds, struggle to understand complex academic course material. This project addresses that challenge by automatically simplifying educational content using AI.

## 🚀 How It Works

1. User provides academic content.
2. The agent analyzes the content.
3. Complex language is identified.
4. The content is rewritten in simpler language.
5. The simplified explanation is returned to the user.
